# GoF Design Pattern

강의를 들으며 훅훅 들리는 패턴들을 알아들을 수 없는 나는 정말 속상해



### 생성 패턴

추상 팩토리(Abstract Factory)

빌더(Builder)

팩토리 메서드(Factory Method)

원형(Prototype)

단일체(Singleton)



### 구조 패턴

적응자(Adapter)

가교(Bridge)

복합체(Composite)

장식자(Decorator)

퍼사드(Facade)

플라이급(Flyweight)

프록시(Proxy)



### 행동 패턴

책임 연쇄(Chain of Responsibility)

명령(Command)

해석자(Interpreter)

반복자(Iterator)

중재자(Mediator)

메멘토(Memento)

감시자(Observer)

상태(State)

템플릿 메서드(Template Method)

방문자(Visitor)
